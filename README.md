# AI Product Description Generator

AI Product Description Generator is a web app that helps ecommerce sellers generate product titles, product descriptions, bullet points, and SEO keywords using Claude AI.

## How It Works

1. The user enters product details.
2. The frontend sends the data to a Next.js API route.
3. The API route sends a structured prompt to Claude.
4. Claude generates the product listing.
5. The generated result is displayed on the page.
